Google Trends Analyzer - Projektbeschreibung

Dieses Projekt ist ein Web-basierter Google Trends Analysator, der mit Streamlit entwickelt wurde. Die Anwendung ermöglicht es Benutzern, Google Trends-Daten für mehrere Keywords zu analysieren und zu visualisieren.

Hauptfunktionen:
- Upload von Excel-Dateien mit Keywords
- Automatische Verarbeitung von Keywords in Batches
- Abruf von Google Trends-Daten für die angegebenen Keywords
- Visualisierung der Trends-Daten in interaktiven Diagrammen
- Export der Daten als CSV-Datei
- Export der Visualisierungen als PNG-Bilder

Verwendung:
1. Bereiten Sie eine Excel-Datei mit einer Spalte namens 'Keywords' vor
2. Laden Sie die Datei über den Datei-Uploader hoch
3. Klicken Sie auf 'Analyze Trends', um die Daten abzurufen und zu visualisieren
4. Laden Sie die Ergebnisse als CSV oder PNG herunter

Technische Details:
- Entwickelt mit Python und Streamlit
- Verwendet pandas für Datenverarbeitung
- Nutzt plotly für interaktive Visualisierungen
- Implementiert Caching für effiziente Datenverarbeitung
- Unterstützt Batch-Verarbeitung für große Keyword-Listen

Die Anwendung ist besonders nützlich für:
- Marktforschung
- SEO-Analysen
- Trendanalyse
- Wettbewerbsbeobachtung
